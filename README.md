# SimpleVPN

SimpleVPM is a minimalistic VPN server written in Go. 

It is not intended to be a full-featured VPN server, but rather an exploration of different VPN Protocols.


## Supported Protocols

- [ ] WireGuard
- [ ] OpenVPN
- [ ] IKEv2
- [ ] L2TP/IPSec
